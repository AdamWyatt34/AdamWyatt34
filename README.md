## .NET Engineer · Azure · AI Engineering

I'm a backend-focused .NET engineer with a passion for building well-structured, maintainable systems. I specialize in **Clean Architecture**, **Domain-Driven Design**, **CQRS**, and **event-driven modular monoliths** — patterns I apply both professionally and across my own projects.

After getting hooked on programming through a single intro course, I've spent the last several years going deep on the .NET ecosystem — from enterprise-scale Azure-hosted applications to hands-on CLI tooling and AI-integrated services. Lately I've been channeling that energy toward **AI engineering**, holding an **Microsoft AI-102 certification** and actively building toward that transition.

When I'm not architecting systems, I'm scaffolding them — I'm currently building **[Modulus](https://github.com/AdamWyatt34)**, a CLI tool that scaffolds opinionated modular monolith solutions with Clean Architecture, DDD, and CQRS baked in from the start.
